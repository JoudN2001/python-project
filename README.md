# Python Fundamentals: University Registration & Billing System

## 📌 Overview
This repository contains two core Python console applications developed to demonstrate a strong understanding of programming fundamentals, data structures, and file handling. 

While I often work with modern web frameworks like Next.js, this project serves as a focused exercise in raw backend logic, conditional validations, and memory management using pure Python.

## 🚀 Projects Included

### 1. University Course Registration System (`register_corse.py`)
A fully functional CLI-based CRUD application simulating a university's registration process.
* **Core Data Structures:** Heavy use of nested `Dictionaries`, `Lists`, and `Sets` to manage state in-memory.
* **Robust Validation:** * Prevents duplicate course registrations.
  * Checks course capacity before allowing enrollment.
  * Validates the existence of student IDs and course codes.
* **Clean UI:** Utilizes Python's f-string formatting to generate perfectly aligned, readable console tables.

### 2. Electricity Bill Calculator (`electricity bill.py`)
A script that calculates tiered utility billing based on consumption brackets.
* **Tiered Logic:** Implements progressive tax and rate brackets using complex `if/elif` structures.
* **File I/O:** Automatically formats and securely appends the generated receipt and customer data to an external text file (`electricity_bills.txt`).

## 🛠️ Technologies & Concepts Used
* **Language:** Python 3 (Vanilla/No external libraries)
* **Concepts:** Data Structures (Sets, Dicts), Control Flow, File Handling (Append mode), String Formatting, Input Validation, Loop State Management.
